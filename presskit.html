Pitch

A never ending freeride under a gentle snow. Beauty and danger of wild nature.
Looks simple, feels real.

Description

Kanyon Karver is a game about snowboarding in a genre of infinite runner. Speed and direction are controlled by turning the board, exactly the same way as it is done in real life. A player can ride downhill, avoid or destroy obstacles and perform various tricks on ramps. Destroying obstacles can initiate an avalanche, which is not easy to escape. All tracks are programmatically generated, which brings natural freeride vibes. Track complexity rises as the player progresses.
Download: https://play.google.com/store/apps/details?id=com.kro13.kanyonkarver
Web demo: https://kro13.itch.io/kk

History

The game development started as a pet-project back in 2019, but got a lot of new features and game mechanics recentlty.
The developer is a snowboarder himself, so the game is truly inspired by real-life experience.

Features

  • Programmaticaly geneated unique tracks.
  • Portrait orientation, can be played with one hand.
  • Unique speed and direction control system.
  • Supports right and left handed control.
  • Quick game sessions.
  • Sessions stored locally: quit and continue from the last point any time.
